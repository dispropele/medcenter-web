# Дополнения к отчёту — печатные документы

## 1.2.2. Описание выходной информации

В раздел 1.2.2 «Описание выходной информации» дополнительно включаются печатные документы. Печатные документы формируются по шаблонам для последующей печати или сохранения в PDF.

- Типы печатных документов:
  - направление на исследования/анализы
  - квитанция/чек об оплате
  - договор/согласие
  - справка/заключение врача
  - фрагменты медицинской карты/формы приёма
  - отчёты по приёмам и услугам

- Формат и обязательные поля (рекомендуется):
  - формат страницы: A4 (210×297 мм), ориентация — портрет/альбом по необходимости
  - формат файла для сохранения/печати: PDF (рекомендуется PDF/A для архивации)
  - обязательные поля: наименование учреждения, логотип, ФИО пациента, дата рождения/ID, дата документа, уникальный номер документа, содержательная часть, подпись/печать, реквизиты оплаты (для квитанций)
  - место для QR/штрих-кода и контактных данных учреждения

- Генерация и выдача:
  - документы генерируются автоматически на основе данных системы по шаблонам
  - доступны для скачивания в PDF и для печати (предпросмотр)
  - электронные копии сохраняются в истории обращения/деле пациента

- Места для макетов (вставьте рисунки самостоятельно):
  - Рисунок A — макет заголовка и реквизитов
  - Рисунок B — макет основной части документа
  - Рисунок C — пример расположения подписи и печати

## 2.4. Форма представления и способы вывода результатов

В раздел 2.4 дополнительно включается способ вывода «Печатные документы» наряду с существующими формами представления.

- Формы представления и варианты вывода:
  - визуализация в интерфейсе (HTML)
  - экспорт в файлы: PDF, CSV, XLSX
  - печатные документы: PDF-шаблоны, готовые для печати

- Способы вывода печатных документов:
  - предпросмотр печати в интерфейсе
  - скачивание PDF-файла
  - печать через браузер (window.print) или отправка на серверный принтер
  - отправка PDF по электронной почте (при необходимости)

- Настройки печати и требования:
  - выбор количества копий, двусторонняя печать, цвет/чёрно-белая печать
  - соответствие шрифтов, отступов и полей для удобочитаемости и юридической значимости
  - обязательные элементы: логотип, реквизиты учреждения, подпись, дата, номер документа

Примечание: макеты и иллюстрации вставьте самостоятельно в отмеченные места.
