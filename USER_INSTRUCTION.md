# 2.9 Инструкция пользователя по работе с программой

## 1. Условия применения системы

Для работы с системой управления медицинским центром «Медцентр» необходимо следующее программное обеспечение:

- **Node.js** (версия 14 и выше) — необходимо установить на сервер;
- **Web браузер** (Google Chrome, Mozilla Firefox, Microsoft Edge или другой современный браузер) — входит в состав операционной системы;
- **Интернет соединение** — для доступа к приложению по локальной сети;
- **Сервер базы данных SQLite** — поставляется в составе приложения и автоматически инициализируется при первом запуске.

## 2. Подготовка системы к работе

### 2.1 Установка и запуск приложения

Перед началом работы с системой на сервере необходимо выполнить следующие действия:

1. **Подготовка окружения:**
   - Убедиться, что на сервере установлен Node.js версии 14 или выше;
   - Распаковать файлы приложения в папку `C:\Program Files\medcenter` или в любую другую удобную папку;

2. **Установка зависимостей:**
   - Открыть командную строку или PowerShell;
   - Перейти в папку приложения: `cd C:\Program Files\medcenter`;
   - Выполнить команду: `npm install`;

3. **Инициализация базы данных:**
   - База данных автоматически создается и заполняется образцом данных при первом запуске;
   - Убедиться, что в папке `db/` появился файл `database.db`.

4. **Запуск приложения:**
   - В командной строке выполнить: `npm start` или `node server.js`;
   - Приложение запустится на порту 3000.

### 2.2 Вход в систему

Для проверки доступности приложения с рабочего места пользователя необходимо выполнить следующие действия:

1. **Открыть приложение:**
   - Открыть любой современный web браузер (Chrome, Firefox, Edge);
   - Ввести в адресную строку: `http://localhost:3000` (если приложение запущено локально) или `http://[IP_СЕРВЕРА]:3000` (для доступа с других компьютеров);
   - Нажать «Enter» или кнопку перехода.

2. **Вход в систему:**
   - На открывшейся странице входа ввести логин и пароль;
   - Нажать кнопку «Вход»;
   - Убедиться, что открылось главное меню приложения (см. Рисунок 2.14).

3. **При первом входе:**
   - Если пользователя еще нет в системе, нажать ссылку «Регистрация»;
   - Заполнить форму регистрации: ФИО, логин, пароль, роль;
   - Нажать кнопку «Сохранить»;
   - После регистрации можно осуществить вход с новыми учетными данными.

## 3. Описание операций

В системе управления медицинским центром реализованы следующие основные операции:

- **Регистрация пользователя** — создание учетной записи для врачей, администраторов и пациентов;
- **Работа со справочниками** — управление специализациями, услугами, видами анализов, врачами и пациентами;
- **Управление назначениями** — создание, изменение и отслеживание приемов пациентов у врачей;
- **Управление контрактами** — оформление договоров с пациентами на оказание услуг;
- **Финансовое управление** — учет чеков, квитанций и расчетов;
- **Печать документов** — вывод направления к врачу;
- **Составление отчетов** — просмотр и печать аналитических отчетов.

### 3.1 Регистрация пользователя

Для регистрации нового пользователя необходимо выполнить следующие действия:

1. На странице входа нажать на ссылку «Регистрация»;
2. Заполнить форму регистрации:
   - **ФИО** — полное имя пользователя;
   - **Логин** — уникальный идентификатор для входа;
   - **Пароль** — установить надежный пароль;
   - **Роль** — выбрать роль (пациент, врач, администратор);
   - Если выбрана роль «врач», указать специализацию;
3. Нажать кнопку «Сохранить»;
4. Система подтвердит успешную регистрацию.

### 3.2 Работа со справочниками

#### Управление специализациями

1. В главном меню выбрать **Администрирование → Специализации**;
2. Для добавления новой специализации:
   - Нажать кнопку «Новая специализация»;
   - Ввести название специализации;
   - Нажать кнопку «Сохранить»;
3. Для изменения специализации:
   - Нажать на название нужной специализации;
   - Отредактировать данные;
   - Нажать кнопку «Обновить»;
4. Для удаления специализации:
   - Нажать на кнопку удаления рядом со специализацией;
   - Подтвердить удаление.

#### Управление услугами

Работа со справочником услуг выполняется аналогично управлению специализациями:

1. Выбрать **Администрирование → Услуги**;
2. Выполнить необходимые операции (добавление, изменение, удаление).

#### Управление видами анализов

Работа со справочником анализов выполняется аналогично:

1. Выбрать **Администрирование → Анализы**;
2. Выполнить необходимые операции.

#### Управление пользователями и врачами

1. Выбрать **Администрирование → Пользователи**;
2. Просмотреть список зарегистрированных пользователей;
3. Для добавления новой записи — нажать кнопку «Новый пользователь» и заполнить форму.

### 3.3 Управление назначениями (приемы пациентов)

#### Создание назначения

1. Выбрать **Назначения → Новое назначение** или нажать кнопку «Новое** на странице списка назначений;
2. Заполнить форму:
   - **Пациент** — выбрать из списка или начать вводить фамилию;
   - **Врач** — выбрать врача из списка;
   - **Дата и время приема** — указать дату и время;
   - **Причина визита** (опционально) — описать жалобы пациента;
3. Система проверит на конфликты времени (перекрытие приемов у одного врача);
4. Нажать кнопку «Сохранить»;
5. Система подтвердит создание назначения.

#### Просмотр и изменение назначения

1. Выбрать **Назначения → Список назначений**;
2. Найти нужное назначение в таблице;
3. Для просмотра деталей — кликнуть на строку назначения;
4. Для изменения — нажать кнопку редактирования и внести необходимые изменения;
5. Нажать кнопку «Обновить**.

#### Печать направления к врачу

1. В списке назначений найти нужное назначение;
2. Кликнуть на кнопку «Печать» или выбрать меню **Действия → Печатный документ**;
3. Откроется форма направления к врачу в новой вкладке браузера;
4. Нажать **Ctrl+P** или выбрать **Файл → Печать** в браузере;
5. Настроить параметры печати и распечатать документ.

### 3.4 Управление контрактами

#### Создание контракта

1. Выбрать **Контракты → Новый контракт**;
2. Заполнить форму:
   - **Пациент** — выбрать пациента;
   - **Дата начала** — дата заключения контракта;
   - **Дата окончания** — дата окончания действия контракта;
3. Нажать кнопку «Сохранить»;

#### Просмотр контрактов

1. Выбрать **Контракты → Список контрактов**;
2. Система выведет таблицу со всеми контрактами;
3. Для просмотра деталей контракта — кликнуть на строку.

### 3.5 Финансовое управление (Чеки и квитанции)

#### Создание чека

1. Выбрать **Финансы → Чеки → Новый чек**;
2. Заполнить форму:
   - **Квитанция** — выбрать квитанцию из списка;
   - **Сумма** — указать размер платежа;
   - **Описание услуги** — выбрать услугу из справочника;
3. Система проверит, что одна услуга не указывается дважды в одной квитанции;
4. Нажать кнопку «Сохранить»;

#### Просмотр квитанций

1. Выбрать **Финансы → Квитанции**;
2. Система выведет список квитанций за выбранный период;
3. Для просмотра деталей — кликнуть на квитанцию.

### 3.6 Составление отчетов

#### Отчет по загруженности врачей

1. Выбрать **Отчеты → Отчет по загруженности и выручке врачей**;
2. (Опционально) Выбрать период:
   - Нажать на поле «Дата начала» и выбрать дату;
   - Нажать на поле «Дата окончания» и выбрать дату;
3. (Опционально) Выбрать фильтры:
   - **Специализация** — фильтр по специализации врача;
   - **Группировка** — выбрать «По специализации» для группировки результатов;
4. Нажать кнопку «Сформировать»;
5. Система выведет отчет в таблице с колонками: ФИО врача, специализация, кол-во посещений, сумма выручки;
6. Для печати отчета нажать **Ctrl+P** или выбрать **Файл → Печать**.

#### Отчет по услугам

1. Выбрать **Отчеты → Отчет по услугам**;
2. Выбрать период (обязательно);
3. (Опционально) Применить фильтры:
   - **Врач** — фильтр по врачу;
   - **Услуга** — фильтр по услуге;
   - **Группировка** — выбрать «По врачу» для группировки;
   - **Сортировка** — выбрать критерий сортировки;
4. Нажать кнопку «Сформировать**;
5. Система выведет отчет;
6. Для печати нажать **Ctrl+P**.

#### Отчет по выданным направлениям

1. Выбрать **Отчеты → Отчет по выданным направлениям**;
2. (Опционально) Выбрать фильтры:
   - **Услуга/Анализ** — фильтр по услуге;
   - **Группировка** — выбрать «По анализам» для группировки;
3. Нажать кнопку «Сформировать**;
4. Система выведет отчет;
5. Для печати нажать **Ctrl+P**.

#### Отчет по назначениям

1. Выбрать **Отчеты → Отчет по назначениям**;
2. Выбрать период (обязательно);
3. (Опционально) Выбрать фильтры:
   - **Группировка** — выбрать «По специализации** для группировки результатов;
4. Нажать кнопку «Сформировать»;
5. Система выведет отчет в таблице;
6. Для печати отчета нажать **Ctrl+P**.

### 3.7 Медицинская карта пациента

1. Выбрать **Пациенты → Медицинская карта**;
2. Начать вводить ФИО пациента или выбрать из списка;
3. Система выведет историю приемов пациента:
   - Назначения (с датами и врачами);
   - Проведенные анализы;
   - Направления;
4. Для просмотра деталей конкретного приема — кликнуть на его строку.

## 4. Сообщения об ошибках и их разрешение

| Ошибка | Причина | Решение |
|--------|---------|---------|
| Не удается подключиться к приложению | Приложение не запущено или недоступен сервер | Убедиться, что приложение запущено; проверить адрес сервера и номер порта |
| Ошибка при входе: неверные учетные данные | Неправильно введен логин или пароль | Проверить написание логина и пароля; перезагрузить страницу браузера |
| Дата начала больше даты окончания | При создании назначения указана неправильная дата | Проверить корректность выбранных дат в форме |
| Врач занят в указанное время | Выбранный врач уже имеет приём в это время | Выбрать другого врача или другое время приема |
| Одна услуга указана дважды | При создании чека указана одна услуга несколько раз | Удалить дублирующиеся записи услуг |

## 5. Советы по работе

1. **Резервное копирование:** Периодически делайте резервные копии файла базы данных `db/database.db`;
2. **Проверка данных:** Перед отправкой на печать проверяйте корректность заполненных данных;
3. **Использование фильтров:** Используйте фильтры и периоды в отчетах для быстрого поиска нужных данных;
4. **Регулярное обновление:** Проверяйте наличие обновлений приложения и устанавливайте их;
5. **Поддержка безопасности:** Используйте надежные пароли и не сообщайте учетные данные другим лицам.
